A Travel Agency Booking System designed to help customers easily explore and book exciting travel packages. Built as a full-stack web application using the MERN stack and Next.js, featuring an intelligent trip planner powered by a fine-tuned **Mistral 7B** AI model with **RAG** (Retrieval-Augmented Generation) to provide accurate, personalized travel recommendations.
